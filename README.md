# Consulta Dietética API

API backend en Django REST Framework para gestionar consultas dietéticas, pacientes, nutricionistas, planes nutricionales y alimentos programados.

## Funcionalidades

- Gestión de pacientes con seguimiento nutricional.
- Registro de nutricionistas y sus honorarios.
- Creación de planes nutricionales con alimentos programados.
- Programación y seguimiento de consultas dietéticas.
- Endpoints REST con paginación, filtros y estadísticas.

## Ejecución

1. Configura las variables de entorno necesarias.
2. Ejecuta migraciones.
3. Inicia el servidor local con Django.
